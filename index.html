Sistema de Administración de Gimnasios

Java / Go / React / SQL

Emprendimiento SaaS Full Stack co-fundado y desarrollado junto a dos socios. Es una solución comercial integral, actualmente hosteada y 100% funcional, diseñada bajo una arquitectura Multitenant para la gestión escalable de múltiples establecimientos.

Implementamos una estructura híbrida que garantiza el aislamiento lógico de datos por cliente, combinando la robustez de Java en el núcleo transaccional con la eficiencia de Go para microservicios, todo integrado en una interfaz reactiva con React.

El sistema centraliza la operación del negocio: gestión de socios, control de accesos en tiempo real y un módulo financiero exhaustivo con cierres de caja y proyección de ingresos. Desplegado en la nube, cuenta con seguridad vía JWT y persistencia optimizada en PostgreSQL.
